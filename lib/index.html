Vuri

A small unobtrusive responsive css grid.

Data Attributes

Vuri leverages HTML's data attributes to make a clear distinction between your styles and the grid. Most grid frameworks need you to mix in class names etc. With Vuri you tag the attribute with a data attribute instead. Like so and vuri makes that element part of the grid
